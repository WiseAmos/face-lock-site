How it works

Three states. That's the whole thing.

PRESENT

You're there

Webcam sees your face and matches it to the enrolled 128-D descriptor. Nothing happens. You're working.

GRACE

You stepped away

Screen dims after 2 s. You have 15 s to come back. If you do, state returns to PRESENT. If you don't, we lock.

LOCKED

OS password

Hard OS lock fires. Only your password comes back. We never auto-unlock after a hard lock — that would defeat the point.

Features

Built for the people who forget to ⌘+⌃+Q.

Matches your face

Not "any face in frame" — a 128-D descriptor generated from your enrollment, matched every 500 ms with a Euclidean distance threshold.

face-api.js · SSD MobileNet · 68-pt landmarks

Liveness check, default ON

Catches printed photos and phone-on-screen attacks. No extra ~5 MB model — uses the texture variance of your face crop plus 1.5 s of landmark jitter.

on by default · no flag exposed · tunable in config

Shoulder-surf dim

Two opt-in triggers: --away-dim for when you turn your head to talk to someone, --multi-face-dim when someone stands behind you.

opt-in · dims only, never locks · off by default

Cross-platform, real installs

macOS, Linux, Windows. Native camera capture via bundled ffmpeg (no system ffmpeg needed). One command installs it as a login service.

macOS · Linux · Windows · Node 18–24

Zero telemetry. Zero network.

Your face descriptor never leaves the machine. It's stored at ~/.face-lock/profile.json (mode 0600). The only network call is a self-update check you can disable.

FACE_LOCK_NO_UPDATE_CHECK=1 to opt out

One short command

Bare facecheck from any terminal — if you have a profile enrolled, it starts the monitor. If you don't, it walks you through setup. Like opencode for screen lock.

facecheck · face-lock · interactive wizard

Install

One line. Then it just runs.

01

Install globally

$ npm install -g face-lock

02

Run the wizard (one time)

$ facecheck

Look at the camera for ~5 s. We capture 3 frames, average the descriptor, save it to ~/.face-lock/profile.json.

03

Install as a login service (optional)

$ facecheck install

Registers a launchd / systemd / Task Scheduler entry that starts the monitor at login. Uninstall with facecheck uninstall.
